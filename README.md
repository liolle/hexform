# Hexform

Full-stack survey platform for rapid creation and distribution of dynamic surveys, inspired by Typeform and Odoo Survey.

## 🚀 Overview

Hexform is a SaaS-style application that allows users to create, share, and analyze surveys with minimal friction.

It focuses on:

* Fast survey creation
* Public/private sharing
* Structured response collection
* Real-time result visualization

## ✨ Features

* Authentication system (user accounts & access control)
* Survey creation with dynamic question types
* Public and private survey sharing via links
* Response collection and storage
* Dashboard with survey analytics and statistics

## 🛠 Tech Stack

* Backend: FastAPI, SQLAlchemy
* Frontend: SolidJS
* Database: Postgresql
